HR Timetable manager Core

This module was written to extend the functionality of Human Resources App to support Schedules and Timetables and allow you to trace working hours

Installation

To install this module, you need to:

  • Just activate module

Configuration

To configure this module, you need to:

  • Get Human Resources Manager access rights
  • Go to Employees menu
  • Enter Schedule submenu and modify or create your own schedule

Usage

To use this module, you need to:

  • Go to specific employee
  • Switch Timetable tab
  • Select schedule for employee
  • Add Timetable line and fill time per day
